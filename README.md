# SOMATIVA

## Como rodar o projeto:


```bash
git clone "link do repositorio"
cd react-senai
cd somativa
```

## Frontend
No primeiro terminal:

```bash
cd front
- npm install
- npm run dev
```

## Backend
No segundo terminal:

```bash
cd back 
- py server.py
```
